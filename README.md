# Tic Tac Toe in React JS
- It's a Multiplayer Tic-Tac-Toe Game made by using ReactJS and Socket.io.

- Clone the Repo and run the following commands in the terminal:
 ```
 - In Backend Folder:
 cd Backend
 npm install
 npm start

------------------

 - In Frontend Folder:
 cd Frontend
 npm install
 npm start
 ```
